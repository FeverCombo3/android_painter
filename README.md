功能描述
------
使用最新的Android知识和框架,展示最新的开发者资讯，自定义View的原理和使用的探索，后期会添加股票软件开发的一些指标等功能。

项目模块：
------


> 首页：首页主要展示Android，IOS，前端等最新资讯，多数为开发者自己的工作经验。
>>数据来源于：[干货集中营](http://gank.io/api)  


> 探索：主要为Android自定义View中的常用API的使用示例。


> 应用：为自定义View方面的尝试，自定义的控件。


> 更多：暂无。
